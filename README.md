This is a simple Telegram Reminder Bot built using Python and the pyTelegramBotAPI library.

## Features
- Sends reminder messages automatically
- Supports multiple users
- Users receive reminders after starting the bot
- Easy to configure reminder time

## Technologies Used
- Python
- Telegram Bot API
- pyTelegramBotAPI

## How It Works
1. Users open the bot and press `/start`
2. The bot saves the user chat ID
3. The bot sends reminder messages at fixed intervals

## Use Case
- Study reminders
- Daily task reminders
- Learning Telegram bot development
